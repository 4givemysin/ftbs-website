## Recommended build order

```
Phase 1  →  Deploy + DNS
Phase 2  →  Replace samples + legal pages + official logo
Phase 3  →  BGW division hub + verticals + gallery
Phase 4  →  Dedicated forms + lead routing
Phase 5  →  Blog, news, resources, SEO strategy (ongoing)
Phase 6  →  Gov portal, PDF, NAICS, SAM, vendor info
Phase 7  →  CRM, CMS, monitoring, CI automation
```

**Parallel opportunities:** Legal copy during Phase 1 deploy · Case study gathering during Phase 2 · SEO strategy draft during Phase 3–4 · SAM/NAICS inventory during Phase 2 certifications
