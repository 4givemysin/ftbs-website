# Phase 6 — Government Contractor Readiness

**Goal:** Full government and vendor procurement readiness with dedicated contracting pages, registrations, and downloadable capability materials.

**Status:** Not started

**Estimated effort:** 4–5 weeks (80–110 hours) + ongoing registration maintenance

**Depends on:** Phases 2 (capability page, certifications, case studies), 3 (BGW), 4 (project inquiry), 5 (resources/SEO)

---

## Business-critical deliverables

| Deliverable | Route (proposed) | Priority |
|---|---|---|
| **Government Contracting Page** | `/government` or `/contracting` | Critical |
| **Capability Statement PDF Download** | `/about/capabilities` + `/resources` | Critical |
| **NAICS Section** | `/government#naics` or `/about/capabilities#naics` | Critical |
| **SAM Registration Section** | `/government#sam` | Critical |

## Task checklist

### Government Contracting Page *(business-critical)*

- [ ] Dedicated hub page for government and institutional audiences
- [ ] Overview: how FTBS and BGW support public sector projects
- [ ] Past performance summary linked to Phase 2 case studies
- [ ] Teaming and subcontracting interest statement
- [ ] RFP/teaming CTA → Project Inquiry Form (Phase 4)
- [ ] Security and data handling overview (technology services)
- [ ] Geographic service area documentation
- [ ] Small business / certification badges (verified only)
- [ ] Print-optimized layout

### Capability Statement PDF Download *(business-critical)*

- [ ] Branded PDF generated from Phase 2 capability web content
- [ ] One-click download from Capability Statement, Government, and Resource Center pages
- [ ] Version number and last-updated date on PDF
- [ ] PDF metadata (title, author, keywords)
- [ ] Optional: BGW-specific capability brief PDF
- [ ] Document owner and quarterly review schedule

### NAICS Section *(business-critical)*

- [ ] Dedicated NAICS section on Government and/or Capabilities pages
- [ ] List primary and secondary NAICS codes with plain-language descriptions
- [ ] Map NAICS codes to FTBS/BGW services
- [ ] Verify codes against current SAM registration
- [ ] Structured data where appropriate

### SAM Registration Section *(business-critical)*

- [ ] SAM registration status (active/in progress — accurate only)
- [ ] UEI display (if registered)
- [ ] Link to SAM.gov entity record (if public)
- [ ] Registration renewal date reminder (internal)
- [ ] Instructions for contracting officers to verify registration
- [ ] Do not display unverified or expired registration claims

### Vendor Information Section *(business-critical)*

- [ ] Vendor information page or section
- [ ] Legal entity name, address, DBA if applicable
- [ ] Primary contact for procurement inquiries
- [ ] Payment/remittance information (as approved for public display)
- [ ] Insurance summary (GL, workers' comp — public-safe)
- [ ] Bonding capacity statement (if applicable)
- [ ] W-9 or vendor packet download (optional, secure delivery)
- [ ] Certifications cross-link to `/about/certifications`

### Compliance & accessibility

- [ ] WCAG 2.1 AA audit with documented results
- [ ] Accessibility statement (`/legal/accessibility`)
- [ ] Section 508 alignment notes
- [ ] Privacy policy update for gov form data and downloads
- [ ] [Cookie Policy](../roadmap/additional-deliverables.md#cookie-policy) compliance
- [ ] [Terms of Service](../roadmap/additional-deliverables.md#terms-of-service) gov-facing review

### Site architecture

- [ ] Footer "Government" or "Contractors" link
- [ ] Downloads section in footer (capability PDF, vendor info)
- [ ] Breadcrumbs on all gov pages
- [ ] Sitemap includes all Phase 6 routes
- [ ] Enhanced Organization JSON-LD (NAICS, UEI, certifications)

### Operational readiness

- [ ] Procurement reviewer walkthrough (contracting officer lens)
- [ ] Quarterly capability PDF and registration review
- [ ] Point of contact for gov inquiries on all relevant pages
- [ ] Internal runbook for RFP document submissions

---

## Completion criteria

Phase 6 is **complete** when all of the following are true:

1. **Government Contracting Page** is live and procurement-review approved.
2. **Capability Statement PDF Download** works from all linked pages.
3. **NAICS Section** lists verified codes mapped to services.
4. **SAM Registration Section** reflects accurate, current status only.
5. **Vendor Information Section** is complete with approved public-safe data.
6. WCAG audit passed; accessibility statement published.
7. Past performance aligns with Phase 2 case studies.
8. Legal and stakeholder sign-off on all registration and capability claims.
