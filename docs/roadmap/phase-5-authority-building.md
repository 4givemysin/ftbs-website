# Phase 5 — Authority Building

**Goal:** Build long-term organic authority through content publishing, resources, and a documented SEO strategy.

**Status:** Not started

**Estimated effort:** 5–8 weeks initial + ongoing (100–160 hours)

**Depends on:** Phases 2–4 (portfolio, forms, BGW hub in place)

**Note:** Projects Portfolio, Case Studies, and Testimonials moved to **Phase 2**. Phase 5 focuses on content marketing and discoverability.

## Task checklist

### Blog *(business-critical)*

- [ ] Create Blog hub (`/blog`)
- [ ] Article template (`/blog/[slug]`)
- [ ] Categories: construction, infrastructure, technology, leadership, BGW
- [ ] Author attribution (President, FTBS team)
- [ ] Related posts and internal linking
- [ ] RSS feed or sitemap inclusion
- [ ] Publish 4–6 founding articles
- [ ] MDX or content file pipeline for maintainability

### News *(business-critical)*

- [ ] Create News hub (`/news`)
- [ ] News article template (`/news/[slug]`)
- [ ] Company announcements, project milestones, partnerships
- [ ] Date-sorted listing with pagination
- [ ] Cross-link news from Home and BGW division
- [ ] Publish 2–3 initial news items
- [ ] Optional: press release PDF downloads

### Resource Center *(business-critical)*

- [ ] Create Resource Center hub (`/resources`)
- [ ] Organized sections: guides, checklists, FAQs, downloads
- [ ] Link capability statement, media kit, and key PDFs (Phase 6)
- [ ] Gated downloads (optional) tied to consultation form
- [ ] Resource categories: Government, BGW, Project Management, Technology
- [ ] Expand FAQ content from Phase 2 into resource articles

### SEO Content Strategy *(business-critical)*

- [ ] Document target keywords by page and service vertical
- [ ] Content calendar (monthly blog + quarterly pillar pages)
- [ ] Service-area landing page plan (when regions confirmed)
- [ ] Internal linking map (hub → spoke model)
- [ ] FAQPage and Article JSON-LD standards
- [ ] Backlink and directory outreach plan
- [ ] Search Console monitoring cadence (weekly/monthly)
- [ ] Competitor and SERP benchmark baseline
- [ ] KPI dashboard: impressions, clicks, conversions by content type

## Completion criteria

Phase 5 **initial release** is complete when:

1. **Blog** is live with at least 4 published articles.
2. **News** section has at least 2 announcements.
3. **Resource Center** is live with organized, downloadable or on-page resources.
4. **SEO Content Strategy** document is approved and calendar started.
5. FAQ expanded; Careers and Media Kit published (if content ready).
6. All content routes appear in sitemap.
7. Search Console baseline established; first monthly SEO report completed.

**Ongoing:** 2–4 blog posts per month; news as events occur; quarterly strategy review.
